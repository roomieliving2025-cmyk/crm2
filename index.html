Instrucciones

1. Idealista → Mis anuncios → Estadísticas → Exportar XLS

Configuración
Conexión Google Sheets
URL Web App
AppScript — Código completo
Pasos para conectar:
  1. Abre sheets.new → crea un Google Sheet nuevo en blanco
  2. En el Sheet: Extensiones → Apps Script
  3. Borra el código que viene → pega el código de arriba (botón "Copiar código")
  4. Guarda (Ctrl+S)
  5. Implementar → Nueva implementación → Tipo: Aplicación web
  6. Ejecutar como: Yo · Quién tiene acceso: Cualquier usuario
  7. Implementar → Autoriza los permisos → Copia la URL que termina en /exec
  8. Pega esa URL arriba en "URL Web App" → Guardar → Probar conexión
⚠ Cada vez que modifiques el AppScript necesitas hacer una nueva implementación (no actualizar la existente) para que los cambios surtan efecto.
